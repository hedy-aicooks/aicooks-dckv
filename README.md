# HEDY Intelligent Equipment - Commercial Kitchen Exhaust Fan Controller Website

## 📁 文件说明

| 文件 | 说明 |
|---|---|
| `index.html` | 主页面（首页、产品、方案、案例、关于、联系我们） |
| `resources.html` | 资料下载页（产品手册、用户指南、认证证书） |
| `styles.css` | 全部样式（深色工业风主题） |
| `script.js` | 交互功能（导航、动画、计数器、表单） |

## 🚀 如何使用（3 种方式）

### 方式一：直接双击打开（最简单）
1. 下载这 4 个文件到同一个文件夹
2. 双击 `index.html` → 浏览器自动打开，网站就能看了

### 方式二：用 VS Code 打开（推荐）
1. 下载安装 [VS Code](https://code.visualstudio.com/)
2. 把 4 个文件拖进一个文件夹
3. 用 VS Code 打开文件夹
4. 安装 Live Server 插件 → 右键 `index.html` → "Open with Live Server"
5. 自动在浏览器打开，改代码后自动刷新

### 方式三：发布到网上（让全球客户访问）
**推荐平台：GitHub Pages（免费）**
1. 注册 [GitHub](https://github.com/) 账号
2. 新建一个仓库（Repository），名字比如 `hedy-website`
3. 把 4 个文件上传到仓库
4. 在仓库设置（Settings）→ Pages → Source 选 `main` 分支
5. 几分钟后，你会得到一个网址，比如 `https://你的用户名.github.io/hedy-website/`
6. 全球任何人都能访问了！

**国内替代方案：**
- **阿里云/腾讯云 对象存储（COS/OSS）**：上传文件，开启静态网站托管，几分钟搞定
- **百度云加速 / Cloudflare**：加速海外访问

## 🎨 配色方案

网站采用**深蓝工业风**，主色调为青蓝色（#00d4ff），体现工业科技感。

如需修改配色，编辑 `styles.css` 顶部的 `:root` 变量：
```css
--primary: #00d4ff;    /* 主色（青蓝） */
--bg-dark: #0a0e17;    /* 背景深蓝黑 */
--bg-card: #151f33;     /* 卡片深蓝 */
--text-main: #e0e8f0;  /* 主文字浅蓝白 */
```
